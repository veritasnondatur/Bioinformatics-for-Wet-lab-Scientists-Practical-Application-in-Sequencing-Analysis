# Bioinformatics for Wet-lab Scientists: Practical Application in Sequencing Analysis
Author: Dr. Vera Laub

This repository contains teaching materials for introductory, hands-on workshops in ChIP-seq and RNA-seq data analysis, designed for researchers with primarily experimental backgrounds.


## The goal of these sessions is to enable participants to:

- understand the structure and challenges of sequencing data

- gain practical experience with common analysis workflows

- develop confidence in exploring and interpreting publicly available datasets


## Contents

- ChIP-seq training slides
Introduction to ChIP-seq data analysis, including peak calling, visualization, and functional interpretation

- RNA-seq training slides
Overview of RNA-seq workflows, including data retrieval, preprocessing, differential expression analysis, and downstream interpretation


## Teaching Approach

The materials follow a hands-on, guided learning format:

- Use of publicly available datasets (links provided in slides)

- Step-by-step walkthroughs of analysis workflows

- Emphasis on practical understanding over theory-heavy instruction

- Designed to support transition from guided exercises to independent analysis


## Target Audience

- Wet-lab biologists with limited computational background

- Early-stage researchers

- Anyone looking for a practical introduction to sequencing data analysis


## Requirements

Basic familiarity with:

- molecular biology concepts

- sequencing technologies (helpful but not required)

- Some exposure to command line or R/Python is beneficial, but not necessary for following the conceptual parts.


## Notes

- These materials were developed and used in academic training settings (e.g. workshops and supervised courses).
They are intended as an introductory resource and can be adapted for different levels of experience.
- Simple .Clarion file is provided for use with WIlsON (http://loosolab.mpi-bn.mpg.de/wilson/). For detailed instructions on how to construct file and navigate WIlsON see https://github.molgen.mpg.de/loosolab/wilson-apps/wiki.


## Learning Outcomes

After completing these materials, participants should be able to:

- navigate publicly available genomics datasets (e.g. GEO)

- understand key steps in ChIP-seq and RNA-seq analysis workflows

- interpret common outputs (e.g. peaks, expression changes)

- identify typical pitfalls and limitations in sequencing data analysis
